# Object Detection Streamlit App

This project is a web application built with **Streamlit** and **YOLOv8** that performs object detection on images and videos.

## Features
- Upload images for object detection
- Upload videos for object detection
- Adjustable confidence threshold
- Option to upload custom YOLO `.pt` weights

## Technologies Used
- Python
- Streamlit
- YOLOv8 (Ultralytics)
- OpenCV
- NumPy
- Pandas
- Pillow

## How to Run Locally
1. Install dependencies
  pip install -r requirements.txt

2. Run the application
   streamlit run app_streamlit.py

## Deployment
This project can be deployed using **Streamlit Community Cloud**.
